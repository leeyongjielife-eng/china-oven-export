# Strategy

## 一句话战略

**一个资源（YUEFU Kamado 工厂）→ 一个市场（Week 2 选定）→ 一个报价（Kamado 陶瓷烤炉出口寻源）**

## 阶段规划

### Phase 0 — 验证期（当前，Week 1-4）

- 目标：**回复**，不是成交
- 指标：100 次外联 → 10+ 回复 → 3+ 深入对话
- 不做：网站建设、复杂运营自动化（CRM/多产品线）、多产品线扩张
- **允许：** W3 外联按日配额本机自动发信（见 [`cofounder.md`](cofounder.md) 发信政策）；不做云端代发 Gmail

### Phase 1 — 首单期（Month 2-3）

- 目标：1 个小订单 OR 1 笔付费寻源服务费
- 指标：完整走通报价 → 付款 → 发货流程
- 重点：供应商核验、证书、包装、物流

### Phase 2 — 聚焦期（Month 4+）

- 目标：重复购买、转介绍
- 只做**回复过且有过对话**的买家类型
- 砍掉无回复的市场和品类

## 竞争定位

### 我们不是什么

- 不是最低价工厂直销噱头
- 不是什么都卖的贸易公司
- 不是假装很大的公司

### 我们是什么

> 帮海外 BBQ 经销商和户外厨具进口商，从佛山 Kamado 工厂采购陶瓷炭烤炉，提供透明 FOB 报价和出口支持，降低采购风险。

## 每周改进循环

```text
研究市场 → 更新 offer → 联系买家 → 记录回复 → 写入 lessons → 下周调整
```

## 风险与应对

| 风险 | 应对 |
|------|------|
| 供应商质量不稳定 | 核验清单 + 小单试单 |
| 买家不回复 | 改 subject line、改买家类型、改市场 |
| 证书不符目的国 | Week 4 前确认目标国认证要求 |
| 运费吃掉利润 | 报价时拆分 FOB + 运费，让买家看清 |
