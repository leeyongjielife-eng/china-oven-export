# Export Basics — 出口基础知识

> Week 1 必读。不懂这些无法报价和沟通。

## 贸易术语（Incoterms）

| 术语 | 含义 | 谁承担运费 | 新手建议 |
|------|------|------------|----------|
| **EXW** | 工厂交货 | 买家全包 | 了解即可 |
| **FOB** | 装运港船上交货 | 买家付海运费 | ⭐ 最常用报价方式 |
| **CIF** | 成本+保险+运费 | 卖方付到目的港运费 | 买家喜欢，你要算清运费 |
| **DDP** | 目的国完税后交货 | 卖方全包 | 新手勿碰 |

**第一阶段报价建议：FOB [港口] USD**

## 关键概念

| 术语 | 解释 |
|------|------|
| **MOQ** | Minimum Order Quantity，最低起订量 |
| **Lead Time** | 从下单到出货的天数 |
| **PI** | Proforma Invoice，形式发票，买家付款依据 |
| **CI** | Commercial Invoice，商业发票，报关用 |
| **PL** | Packing List，装箱单 |
| **BL** | Bill of Lading，提单 |
| **HS Code** | 海关编码，烤炉约 8514 或 8417 类（需核实具体型号） |
| **LCL** | 拼柜，货少时按体积计费 |
| **FCL** | 整柜，20GP / 40HQ |

## 付款方式

| 方式 | 风险 | 建议 |
|------|------|------|
| T/T 30% deposit + 70% before shipment | 中 | ⭐ 首单推荐 |
| L/C 信用证 | 低但复杂 | 大单再用 |
| 100% 预付 | 买家不接受 | 别要求 |
| OA 账期 | 卖方风险高 | 老客户再说 |

## 烤箱出口 HS 编码（需核实）

- 电热烤炉：通常 8514.40 或类似
- 烘焙烤炉：可能 8417.80

**报价前用 HS 编码确认退税率和监管条件。**

## 运费怎么算（简化）

```text
买家到岸成本 ≈ FOB 价 + 海运费 + 目的港费用 + 关税 + 增值税
```

报价时至少拆分：**FOB 价** 和 **运费估算** 分开写，让买家看清。
