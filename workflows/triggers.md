# 触发词速查 — 忘了就说「速查」

> 收藏本文件，或对话里说 **`速查`** / **`触发词`**，Agent 会贴出此表。

---

## 日常（不用记，默认中等模型即可）

```text
继续推进 / 下一步干嘛 / 写邮件 / 找买家 / 询价草稿
→ 无需触发词，窗口保持 gpt-5.5-medium
```

---

## 需要升高级时 — 复制这句话

| 你想做 | 直接说（复制） | 切什么模型 |
|--------|----------------|------------|
| 每周战略复盘 | **`Founder复盘`** | Opus |
| 选定第一个出口市场 | **`正式Research`** | Grok |
| 报价终稿发给买家前 | **`正式报价`** | Codex |

---

## 其他常用启动词

| 你说 | 发生什么 |
|------|----------|
| `速查` / `触发词` | 弹出本表 |
| `启动 Research` | 轻量调研（默认模型） |
| `启动 Sales` | 找买家 / 写外联 |
| `启动 Supplier` | 供应商核验 / 询价草稿 |

---

## Founder Agent 主动提醒规则

以下情况 **Agent 必须主动提醒**触发词，不用你记：

| 你说了类似的话 | Agent 应提醒 |
|----------------|--------------|
| 「这周总结一下」「复盘一下」 | 💡 说 **`Founder复盘`** 可升 Opus 做深度复盘 |
| 「选哪个国家」「定市场」「研究市场」 | 💡 说 **`正式Research`** 可升 Grok 联网选市场 |
| 「发正式报价给买家」 | 💡 说 **`正式报价`** 可升 Codex 校对数字 |
| 「忘了怎么说」「触发词是什么」 | 直接贴本速查表 |

---

## 窗口模型速记

```text
平时  →  gpt-5.5-medium（或 composer-2.5-fast）
复盘  →  切 Opus + 说 Founder复盘
选市场 →  说 正式Research（建议开 Task）
```
