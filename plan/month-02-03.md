# Month 2-3 — 首单

## Month 2 重点

**只跟进回复过的买家。不撒大网。**

### 每周任务

- [ ] 跟进所有 Qualified / Quoted 买家
- [ ] 对每个活跃买家确认：
  - 烤炉类型、尺寸
  - 电压 / 燃气
  - 目的港
  - 预算范围
- [ ] 根据需求匹配供应商，48 小时内回报价
- [ ] 记录买家犹豫点 → `lessons.md`
- [ ] 每周 1 次市场研究微调（`research/weekly-prompt.md`）

### Month 2 指标

| 指标 | 目标 |
|------|------|
| 活跃对话 | 3+ |
| 正式报价 | 3+ |
| 验货/样品讨论 | 1+ |

---

## Month 3 重点

**目标：1 个小订单 OR 1 笔付费寻源服务。**

### 可能的「首单」形式

1. **1-2 台烤炉试单** — 最常见，金额小、流程完整
2. **付费寻源报告** — 收固定 fee，帮买家核验 + 报价 3 家供应商
3. **样品单** — 买家付样品费 + 运费

### 成交前 Checklist

- [ ] 买家需求书面确认（邮件即可）
- [ ] 供应商核验完成
- [ ] 合同或 PI（Proforma Invoice）双方确认
- [ ] 付款方式明确（建议 T/T 30/70）
- [ ] 交期书面确认
- [ ] 出口文件清单确认
- [ ] 包装要求确认

### 若 Month 3 仍无单

回 `lessons.md` 复盘：

1. 市场选错？→ 换国家
2. 买家类型错？→ 换分销商/餐厅/批发商
3. 外联文案差？→ A/B 测 subject line
4. 价格无竞争力？→ 换供应商或调 markup
5. 缺证书？→ 优先解决认证

**不要加产品线，先修一个环节。**
