# Research 工作区索引

> Research 子代理的**操作数据**放这里；**结论摘要**写回 `company/research.md`。

```text
markets/
├── README.md                 # 本文件
├── country-scoring.md        # 国家评分表（正式Research 填）
├── research-playbook.md      # 分步调研手册（WHO/WHERE/PAIN）
├── competitor-benchmark.md   # 竞品与价格锚点
├── keywords.md               # 20 个搜索关键词
└── candidates/               # 各国原始笔记（可选）
    └── README.md
```

## 研究三阶段

| 阶段 | 何时 | 模型 | 产出 |
|------|------|------|------|
| **A 设计** | 现在 | 默认 | 框架、模板、候选国清单 |
| **B 轻量验证** | Week 2 Day 1-2 部分 | 默认 | 假设、初步笔记 |
| **C 正式选市场** | Co-founder 说 `正式Research` | Grok 高级 | 填满 scoring + 写回 company/ |

## 完成检查（正式 Research 后）

- [ ] `country-scoring.md` — 至少 5 国打分，Top 3 明确
- [ ] `competitor-benchmark.md` — BGE/KJ 零售价 + 1 家中国供应 Alibaba 价
- [ ] `keywords.md` — 20 词填完，按选定国家分组
- [ ] `company/research.md` — Top 3 国、买家类型、niche
- [ ] `company/customers.md` — **第一个目标市场** + 3 条理由
